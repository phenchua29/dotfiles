# Obsidian Vault Rules Skill

**Before creating or editing any note, you MUST read the vault's canonical Rules folder.**
**It is the single source of truth - do not rely on anything hardcoded here.**
* Locate and read these files from the `Rules` folder within the active vault or workspace root.
* Do not read files from any absolute paths outside the current workspace.

Read all three files before proceeding:

- [frontmatter-schema.md](Rules/frontmatter-schema.md)
- [type-values.md](Rules/type-values.md)
- [area-values.md](Rules/area-values.md)
